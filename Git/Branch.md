#### List all branch
```bash
git branch
```
### Load a branch
```bash
git switch <name_branch>
```
### Create a branch
- without move onto 
```
git branch <name-of-branch>
```
- with move onto
```
git branch <name-of-branch>
git checkout <name-of-branch>
```
ou
```bash
git checkout -b <name_new_branch>
```
